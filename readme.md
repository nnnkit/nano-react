# Nano React Boilerplate

Step 1: Clone this repo
```
git clone --depth=1 git@github.com:nnnkit/nano-react.git sample
cd sample
```

Step 2: Install
```
npm install
```

Step 3: Start

```
npm run dev
```

### Includes

* React 16.8
* ReactDOM
* SCSS
* ES6 / ES7
* Proposal Class Properties
* Webpack
* Babel
